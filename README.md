# ⚡ CodeSpark - AI Developer Assistant

**CodeSpark** es un asistente inteligente para desarrolladores diseñado para optimizar el flujo de trabajo mediante el uso de Inteligencia Artificial (Google Gemini). Permite explicar fragmentos de código, generar pruebas unitarias y mejorar la calidad del texto o código de manera instantánea.

## 🚀 Características Principales

* **👨‍🏫 Code Tutor:** Explica lógica compleja de programación en lenguaje sencillo.
* **🧪 Test Generator:** Crea casos de prueba automáticos para asegurar la robustez del código.
* **✍️ Smart Editor:** Mejora y refina fragmentos de código o documentación técnica.
* **📜 History Tracking:** Guarda tus consultas recientes localmente para un acceso rápido.
* **🌙 Dark Mode Interface:** Diseñado para programadores, con una estética moderna en tonos oscuros y acentos naranja.

---

## 🛠️ Tecnologías Utilizadas

### Frontend
* **Angular 18/19:** Framework principal.
* **TypeScript:** Tipado fuerte para un desarrollo seguro.
* **CSS3 Custom Properties:** Para el sistema de temas (Dark Mode & Spark Orange).

### Backend
* **Node.js & Express:** Servidor de API REST.
* **Google Generative AI SDK:** Integración directa con el modelo Gemini Pro.
